# Deploy Klasörü

Bu klasör Plesk kurulumu için Nginx konfigürasyonlarını içerir.

| Dosya | Kullanım |
|-------|----------|
| `plesk-nginx.conf` | **Kullanın.** Proxy mode AÇIK iken sadece WebSocket bloğunu yapıştırın |
| `plesk-nginx-full.conf` | Proxy mode KAPALI ve `location /` yoksa tam konfig için |

**Ana kurulum rehberi:** Proje kökündeki [KURULUM.md](../KURULUM.md)
